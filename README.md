# fishing
ゲーム仕様書：フィッシング・エボリューション（Ver. 3.0）

1. 企画概要

コンセプト: 魚を釣り、売り、装備を強化する「成長型釣りゲーム」

操作: クリック / タップのみ（PC・スマホ両対応）

主要機能: ランクと星による成長システム、スロット制スキル付与、高精度セーブ機能

2. 釣り竿・強化システム

2.1 釣り竿のランク

釣り竿を買い換えることでベースパワーが大幅に上昇。

名前例：木の釣竿、竹の釣竿、カーボン釣竿、チタン釣竿、伝説の釣竿。

2.2 星（スター）とスキルスロット

星の数 (0〜5): 各竿ごとに強化可能。

パワー上昇: 星1つごとにパワーが一定値上昇。

星4つのパワー ≒ 次のランクの竿（星0）のパワー となるよう調整。

スキルスロット: 竿に付いている「星の数」と同じ数だけスキルを装着可能。

例：星2の竿 ＝ スキルを2つまで装着できる。

3. ゲームサイクルと判定

3.1 釣りの流れ

キャスト: クリックでウキを投げる。

待機: 魚がつつくとウキが揺れる予兆。

ヒット: ウキが完全に沈んだ瞬間にクリック。

パワー判定:

自分の釣りパワー >= 魚のパワー: 即座に釣り上げ成功。

自分の釣りパワー < 魚のパワー: ゲージバトルへ移行。

3.2 ゲージバトル（キャッチング）

針（インジケーター）が左右に高速往復する。

赤（大成功）: 捕獲確率 80%〜100%

緑（成功）: 捕獲確率 40%〜60%

白（普通）: 捕獲確率 10%〜20%

自分のパワーが魚のパワーに近いほど、ゲージの動きが遅くなり、赤色の範囲が広くなる。

4. ショップと経済

魚の売却: インベントリ内の魚をすべて現金化。

釣り竿の購入: 新しいランクの竿をアンロック。

釣り竿の強化: お金を消費して「星」を増やす（最大5）。

スキルの購入: 装着可能なスキル（パワーUP、ゲージ減速、価格UP等）を購入。

餌の購入: 魚がヒットするまでの時間を短縮する。レアな魚が出やすい

5. 永続化（セーブ・ロード）要件 ※重要

「後日アプリを開いても続きから遊べる」ことを保証するため、以下の仕様を実装する。

5.1 保存タイミング（オートセーブ）

魚を釣り上げた瞬間

ショップで売却・購入・強化を行った瞬間

スキルを付け替えた瞬間

原則：全ての「変化」が発生した直後にバックグラウンドで保存を実行する。

5.2 保存方法

ブラウザを閉じたり、PCを再起動してもデータが保持される。

5.3 保存データ構造（例）

{
  "version": "1.0",
  "saveDate": "2024-05-20T12:00:00Z",
  "player": {
    "money": 1500,
    "baitCount": 10
  },
  "rod": {
    "rankIndex": 1,
    "stars": 3,
    "equippedSkills": ["power_up_1", "gauge_slow_1", "none"]
  },
  "unlocked": {
    "rods": [0, 1],
    "skills": ["power_up_1", "gauge_slow_1"]
  }
}


5.4 ロード処理

スタート画面で「つづきから」を押した際、上記データを読み込み、各ステータス（所持金、竿の見た目、星の数、スキル効果）を完全に復元する。

6. 画面設計

スタート: 背景に波の音。大きな「START」ボタン。

釣り画面: 1人称視点。スマホでも押しやすいよう画面全体がクリック判定。

ショップ: リスト形式でアイテムを配置。現在の竿の「星」と「スロット」が視覚的にわかるUI（★マーク表示など）。
