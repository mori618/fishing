# フィッシング・エボリューション (Fishing Evolution) Ver 3.0

魚を釣り、売り、装備を強化していく「成長型釣りゲーム」です。
クリック（タップ）のみの簡単操作で、本格的な釣りアクションとRPG的な成長要素が楽しめます。

## 1. 企画概要・コンセプト

*   **ジャンル**: シミュレーション / アクション
*   **プラットフォーム**: Webブラウザ（PC・スマートフォン両対応）
*   **操作**: マウス操作（クリック）、タッチ操作（タップ）のみ
*   **主要機能**:
    *   **ランク・星システム**: 釣り竿のランクアップと、星（スター）による強化
    *   **スキルビルド**: 竿の星の数だけ解放されるスロットに、自由にスキルを装着
    *   **図鑑コンプリート**: 全35種の魚と、激レアな「称号付き（ヌシ）」の収集
    *   **オートセーブ**: 釣果や購入状況を即座に保存・復元

## 2. ゲームサイクル

1.  **Fishing (釣り)**: 魚を釣ってインベントリに溜める。
2.  **Selling (売却)**: ショップで魚を売却し、お金（G）を稼ぐ。
3.  **Upgrade (強化)**: 新しい竿の購入、竿の強化、スキルの購入・装備を行う。
4.  **Loop**: 強化された装備で、より強力で高価な魚（上位ランク・ヌシ）に挑む。

## 3. 釣りシステム詳細

### 3.1 釣りの流れ
1.  **キャスト**: 画面タップでウキを投げる。
2.  **待機 (Waiting)**: 魚がかかるのを待つ（数秒）。スキルで短縮可能。
3.  **予兆 (Nibble)**: ウキがピクピクと動く（通常3〜6回）。
4.  **ヒット (Hit)**: ウキが完全に沈んだ瞬間にタップ！
    *   **重要**: レア度が高い魚ほど、ヒット受付時間（Hit Window）が極端に短くなります。
    *   **Dランク**: 800ms
    *   **Cランク**: 600ms
    *   **Bランク**: 500ms
    *   **Aランク**: 400ms
    *   **Sランク**: **300ms** (一瞬の遅れで逃げられます)
5.  **パワー判定**:
    *   `自分のパワー` >= `魚のパワー`: **即釣り上げ成功！**
    *   `自分のパワー` < `魚のパワー`: **ゲージバトル**へ移行。

### 3.2 ゲージバトル (Catching)
パワーが足りない場合、テクニックでカバーするモードです。
インジケーターが左右に往復するので、タイミングよく停止させます。

*   **ゲージ速度**: パワー差が大きい（無謀な挑戦）ほど高速になります。
*   **ゾーン判定**:
    *   🔴 **赤ゾーン (Perfect/大成功)**: 捕獲率 **80%〜100%**
    *   🟢 **緑ゾーン (Good/成功)**: 捕獲率 **40%〜60%**
    *   ⚪ **白ゾーン (Normal/普通)**: 捕獲率 **10%〜20%**
*   **赤ゾーン幅**: パワー差が小さい（惜しい）ほど広くなり、スキルで拡張可能です。

### 4.2 星（スター）強化
各竿は最大 **5つ** まで「星」を強化できます。
*   **パワー上昇**: 星1つごとにパワーボーナス（竿ランクにより+3〜+20）が加算されます。
*   **スキルスロット**: **星の数 = 装備できるスキルの数** です。
*   星4まで強化した竿のパワーは、次のランクの竿（星0）とほぼ同等になります。

#### 星強化（スターアップグレード）費用テーブル
各ランクの竿を強化する際の費用です。星4以降の費用を、次のランクの竿の本体価格より高く設定することで、「無理に上位に挑むか、今の装備を完璧にするか」のジレンマを生みます。

| C (竹) | 500 G | 1,200 G | 2,500 G | 5,000 G | 10,000 G |

## 5. スキルシステム (Skills)
ショップで購入し、スロットに装備することで効果を発揮します。

### パワー強化系
*   **パワーUP I〜III**: 釣りパワーを直接底上げ（+5 〜 +30）。即釣り上げラインを引き下げます。

### ゲージ対策系
*   **ゲージ減速 I〜II**: バトル時のゲージ速度を低下（-10% 〜 -20%）。
*   **テクニシャン I〜III**: 赤ゾーン（大成功）の幅を拡大（+20% 〜 +60%）。
*   **起死回生 I〜III**: 白ゾーン（失敗）でも確率で成功扱いに書き換える（10% 〜 35%）。

### 効率・利益系
*   **売値UP I〜II**: 魚の売却価格をアップ（+10% 〜 +25%）。
*   **キャッチ率UP I〜II**: バトル時の基礎捕獲率を底上げ（+5% 〜 +15%）。
*   **忍耐力 I〜III**: 待ち時間を短縮（-10% 〜 -40%）。
*   **餌の達人 I〜III**: 釣り成功時に確率で餌を消費しない（15% 〜 50%）。

### レア狙い系
*   **レア魚UP I**: レア魚の出現率アップ (+20%)。
*   **鑑定眼 I〜III**: **「称号付き（ヌシ）」** の出現率を数倍にする（2倍 / 3倍 / 4倍）。
*   **大物狙い I〜III**: 自分より格上のランクの魚が出やすくなる（出現倍率：1.5倍 / 2.5倍 / 5.0倍）。
*   **予兆察知**: ウキの予兆回数が必ず「2回」で固定される（タイミングが読みやすくなる）。
*   **集中力 I〜III**: シビアなHIT受付時間を延長する（1.5倍 〜 3倍）。
*   **トレジャーハンター I〜III**: 宝箱の出現確率をアップさせる（+2% 〜 +10%）。
*   **フォーチュンハンター I〜III**: 宝箱の報酬量（金・餌・スキル数）をアップさせる（+20% 〜 +100%）。
*   **ラグジュアリーハンター I〜III**: 宝箱の質（金・餌ランク・スキルレア度）をアップさせる（小 〜 大）。

## 6. 宝箱システム (Treasure Chests)
釣り場では、魚の代わりに稀に「宝箱」が釣れることがあります。

*   **出現確率**: 基本 5%（スキルで強化可能）
*   **中身**: お金、餌、スキルが含まれます。高ランクの宝箱ほど、より高価な報酬や強力なスキルが出現しやすくなります。

## 7. ガチャシステム (Gacha)
Ver 3.0より、スキルをお得に獲得できるガチャシステムが導入されました。
ショップ画面の「ガチャ」タブから利用可能です。

### 6.1 常設ガチャ
3種類のガチャがあり、それぞれ出現するスキルのレアリティ傾向が異なります。
10連ガチャは、単発10回分よりもお得な価格設定になっています（1回分無料など）。

| ガチャ名 | 単発価格 | 10連価格 | 特徴 |
| :--- | :--- | :--- | :--- |
| **ブロンズ** | 1,000 G | 9,000 G | 初心者向け。Tier 1スキルが中心 (85%)。 |
| **シルバー** | 8,000 G | 72,000 G | 中級者向け。Tier 2スキルが狙いやすい (75%)。 |
| **ゴールド** | 30,000 G | 250,000 G | 上級者向け。Tier 3スキルが高確率 (75%)。 |

### 6.2 リサイクルガチャ (エコ・ボックス)
不要になったスキルを有効活用するシステムです。
**任意のスキル5個** を消費することで、新しいスキル1個に交換（リサイクル）できます。

*   **特徴**: 通常のガチャよりも少し良い確率設定になっています。
*   **Tier 4チャンス**: わずかですが、最高ランク（Tier 4/Special）が出現する可能性があります (1%)。
*   **排出率**: Tier 1 (40%) / Tier 2 (50%) / Tier 3 (9%) / Tier 4 (1%)

## 9. 図鑑・コレクション要素

### 魚図鑑
釣った魚は図鑑に記録されます。ランクごとのコンプリートを目指しましょう。

### 称号付き（ヌシ）システム
すべての魚種数に対して、約 **1.5%** の確率で「称号付き（Unique/Master）」が出現します。
*   **特徴**: 名前の前に「水たまりの」「深海よりの」などの二つ名が付きます。
*   **価値**: 売却価格が通常種の **10倍** になります。
*   **記録**: 図鑑には通常種とは別に記録され、金色の文字で表示されます。
    *   **隠しボーナス**: 高価なランクの餌を使い、運悪く低いランクの魚が釣れてしまった場合（例：B餌でDランク、A餌でCランクなど）、救済措置としてこの「ヌシ」出現率が **3倍** に跳ね上がります。
